
# Fullstack GraphQL

Simple Demo Application

**API** built with Node + Express + GraphQL + Sequelize (supports MySQL, Postgres, Sqlite and MSSQL). 

**WebApp** built with React + Redux. 

Written in ES6 using Babel + Webpack.
